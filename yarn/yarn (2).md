# Yarn 新一代的包管理工具

```
  yarn eject
	  暴露配置 Y
	  自动多出 config, scripts

  yarn add less@^3.0.3
    3 主版本
    0 次版本
    3 打补丁的版本
```


## Yarn的优点
  速度快
  安装版本统一，更安全
  更简洁的输出
  更好的语义化
  Yarn是相对于 npm而言


## yarn 命令
```
yarn / yarn install 初始化项目

yarn add jquery 安装包
yarn add jquery --save 安装项目依赖
yarn add jquery --dev 安装开发依赖

yarn remove jquery 卸载包

yarn upgrade 升级全部

yarn upgrade jquery 升级 jQuery

yarn global add jquery 全局安装 jquery

```
