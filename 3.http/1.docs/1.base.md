### 1.2 HTTP 协议发展历史
#### 1 http/0.9

   - 只有一个命令 get
   - 没有 header 等描述数据的信息
   - 服务器发送完毕，就关闭 TCP 链接

#### 2 HTTP/1.0

   - 增加了很多命令
   - 增加了 status, code, header
   - 多字符集支持，多部分发送，权限，缓存等

#### 3 HTTP/1.1

   - 持久链接
   - pipeline
   - 增加了 host 和其他一些命令

#### 4 HTTP/2

   - 所有数据以 二进制传输，之前版本是以 字符串进行传输
   - 同一个链接里面发送多个请求，不再需要按照顺序来
   - 头信息压缩以及推送等提高效率的功能
